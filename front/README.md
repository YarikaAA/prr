# Anonymous Feedback Platform

[Read Blog](https://unclebigbay.com/introducing-the-anonymous-feedback-platform-powered-by-the-chern-stack)



## Technologies
- Yarn
- React JS
- Html
- Css
- Bootstrap

## Features
- 👨‍ Personal Account 
- 🔗 Unique Feedback Link 
- 🤩 Receive Feedback with Emojis 
-  ⚙ Manage Feedbacks 
- ⭐ Starred Feedback
